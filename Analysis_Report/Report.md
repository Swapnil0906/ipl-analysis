# 📊 IPL Data Analysis Report

---

## 🧾 Current Version
This project performs exploratory data analysis (EDA) on IPL ball-by-ball data.

### Features:
- Team performance analysis
- Over-wise scoring trends
- Pressure-based analysis using required run rate

### Limitations:
- No machine learning model yet
- No player-level deep analysis
- No visualization dashboard

---

## 📊 Analysis So Far

### 1. Team Performance
Mumbai Indians, CSK, and KKR have the highest total runs, indicating long-term consistency and strong batting lineups.

---

### 2. Over-wise Scoring
Run rates increase significantly in the final overs (16–20), showing aggressive batting strategies during death overs.

---

### 3. Pressure Analysis
- Low pressure → higher scoring
- Medium pressure → cautious play
- Very high pressure → aggressive hitting

This shows player behavior changes based on match situation.

---

## 🧠 Key Learnings
- Data needs cleaning before analysis
- Raw data is hard to interpret without grouping
- Cricket strategy impacts statistical patterns

---

## 🔮 Future Possibilities

- Build ML model to predict runs
- Predict match outcomes
- Player performance analysis

---

## 📌 Conclusion
This project provides foundational insights into IPL match dynamics and sets the stage for predictive modeling.
