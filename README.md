# IPL Strategic Insights & Performance Analysis

A comprehensive analytical study of Indian Premier League (IPL) delivery-level data to quantify team dynamics, scoring patterns, and situational performance metrics.

## 📋 Project Objective
This repository houses a systematic Exploratory Data Analysis (EDA) of IPL history. The goal is to move beyond basic statistics to identify high-leverage patterns that influence match outcomes, providing a data-driven foundation for future predictive modeling.

## 🛠️ Tech Stack
- **Language:** Python 3.x
- **Libraries:** Pandas (Data Manipulation), Matplotlib/Seaborn (Visualization), NumPy (Numerical Computing)
- **Environment:** Jupyter Notebooks

## 📂 Project Structure
- `data/`: Contains `ipl_ball_by_ball_delivery.csv`.
- `notebooks/`: Modular notebooks for data cleaning and feature engineering.
- `reports/`: In-depth technical analysis and conclusions.
- `src/`: Reusable Python modules for data processing.

## 📈 Analytical Highlights
- **Phased Scoring Trends:** Quantified the acceleration delta between Powerplays and Death Overs.
- **Pressure Metrics:** Analysis of Required Run Rate (RRR) as a variable for batting aggression.
- **Consistency Mapping:** Long-term performance auditing of franchise powerhouses (MI, CSK, KKR).
